### How to configure system for run tests:
1. Install Java SE, Android SDK.
1. Install Genymotion.
1. Install Appium.
1. Set ANDROID_HOME and JAVA_HOME environment variables.
1. Add device to Genymotion.
1. Install on device Genymotion_ARM_Tranclation.
1. Install on device gApps.
1. Correct your deviceName and uuid in method "prepareAndroidAppium".
1. Run tests.
